# Calculator-AVR
Make Calculator on Proteus by AVR ( ATMEGA32 ) and Display 16 Bit

# Options:
- 16-digit number support
- Submission
- Multiplication
- Division 

# Requirements: 
- ATMEGA 32 OR 16
- Display 16 bit
- Matrix Key Board
